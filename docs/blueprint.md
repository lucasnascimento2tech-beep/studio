# **App Name**: Guia 2tech

## Core Features:

- Visão Geral da Jornada: Painel central que apresenta o progresso total e o status das fases, do início até a Operação Assistida.
- Entrega de Conteúdo e Materiais: Interface para visualização de vídeos rápidos e materiais de leitura organizados por módulos.
- Envio de Evidências Simulado: Ferramenta para anexar arquivos e capturas de tela obrigatórios para comprovação de tarefas concluídas.
- Checkpoint com Quizzes: Questionários de validação por fase com trava lógica que exige acerto de 70% para avanço.
- Agendamento Inteligente: Liberação condicional de links de reunião com o implantador somente após a conclusão dos pré-requisitos da fase.
- Portal do Implantador: Interface administrativa simplificada para que especialistas aprovem evidências e liberem fases subsequentes.
- Persistência em Estado Local: Armazenamento do progresso, respostas e arquivos simulados utilizando LocalStorage para simular um banco de dados.

## Style Guidelines:

- Cores Principais: Azul 2tech (#004B84) como cor primária, Amarelo (#FFA81F) para destaques e botões de agendamento.
- Cores Neutras: Cinza claro (#F1F1F1) para o fundo da aplicação e Branco (#FFFFFF) para os cards de conteúdo.
- Texto: Azul Escuro (#0A1628) para títulos principais e cinza médio (#667085) para textos secundários.
- Combinação de fontes: 'Poppins' para títulos e manchetes modernas, pareado com 'Inter' para o corpo de texto e informações detalhadas.
- Uso de ícones minimalistas de linha para identificar categorias como Material, Checklist e Encontros Guiados.
- Design de Grid flexível em estilo Dashboard premium B2B, focado em cards expansíveis, bordas arredondadas e sombras suaves.
- Transições suaves entre estados de progresso e animações discretas ao desbloquear novas fases.
